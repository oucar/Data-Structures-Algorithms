## Array ADT 

* ADT = Representation of data + Operations
  *  Append - at the end of the list. 
  *  Insert - to a given index.
```cpp
/*      1      */
int A[10]; //Size = 10, Length = 10

/*      2      */
int *B;
int size;
B = new int[size]
```
